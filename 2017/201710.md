# knowledge
### 一、单行文本溢出显示省略号

``` bash
.ellipsis{
  overflow: hidden;
  white-space: nowrap;
  text-overflow: ellipsis;
}
```

### 二、多行文本溢出显示省略号
>WebKit浏览器或移动端的页面

``` bash
.ellipsis{
  overflow : hidden;
  text-overflow: ellipsis;
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
}
```
